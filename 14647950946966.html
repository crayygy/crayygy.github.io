如何在断开ssh 时保持服务器程序运行？

6/1/2016 posted in  Server  

How to keep server progress running after ssh disconnected?

My answer is:

Screen

00x0

I start to think about this cause I am running a rails server during my work, while connected using ssh, I have to keep my ssh connection.
And this is really annoying.

So I started to google and read blogs about server progress, finally I found Screen.

00x1 install

Ubuntu > sudo apt-get install -y screen

CentOS > yum install -y screen

00x2 usage

Using screen command to open a screen

 $ screen

the run a progress like 

$ rails server

After running a progress, using Ctrl-a then d to detach

list all exist screen sessions: 

$ screen ls

resume screen

$ screen -r

if you have only one screen background, this will bring you back, or gives you a list of screen sessions with pid like 

8858.pts-0.iZ28*****

you can resume a specified screen with 

$ screen -r 8858.pts-0.iZ28

If you need to step away from your computer for a minute, you can lock your screen session using Ctrl-a then x. This will require a password to access the session again.

If you want to close a screen session, just using Ctrl-a then k to shut it down.

To get more information, you can run 

$ man screen

00x3

After running a progress in screen, you can disconnect your ssh without any trouble.
